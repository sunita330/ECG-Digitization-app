Processing Pipeline

Six-Stage Signal Recovery

01

Grayscale Conversion

Eliminates colour noise using ITU-R BT.601 luminance weighting.

02

Otsu Thresholding

Automatic global binarization minimising intra-class variance.

03

Grid Removal

Directional morphological opening with H/V structuring elements.

04

Waveform Enhancement

Dilation and closing reconnect gaps and smooth boundaries.

05

Signal Extraction

Column-wise centroid maps pixels to (x, y) amplitude samples.

06

ECG Plotting

Renders publication-quality signal graph with amplitude calibration.
